# Kanji Info Popover Configuration

`displayConfig` - what is displayed inside the popup:

- `displayKanji` - should Kanji be displayed
- `displayMeanings` - should English meanings be displayed
- `displayOnyomi` - should Chinese readings be displayed
- `displayKunyomi` - should Kunyomi be displayed
- `displayHeisig` - should Heisig English word be displayed
- `displayGrade` - should Grade be displayed
- `displayJLPT` - should JLPT level be displayed
- `displayStrokeCount` - should stroke count be displayed
