# Kanji Info Popover Configuration

`displayConfig` - what is displayed inside the popup:

- _displayKanji_ - should Kanji be displayed
- _displayMeanings_ - should English meanings be displayed
- _displayOnyomi_ - should Chinese readings be displayed
- _displayKunyomi_ - should Kunyomi be displayed
- _displayHeisig_ - should Heisig English word be displayed
- _displayGrade_ - should Grade be displayed
- _displayJLPT_ - should JLPT level be displayed
- _displayStrokeCount_ - should stroke count be displayed

_shouldUseAPI_ - when set to _false_ add-on will not fetch unknown Kanji from API
